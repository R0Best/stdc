# stdc

An extensive std library for C with a focus on safety, speed, and ease of use. It facilitates assistance in memory management, data structures / containers, algorithms, string manipulation, file I/O, console I/O, networking and other while having no dependencies (this includes libc) and being cross platform. It is also designed to be easily extendable by users and has a modular design. The library is written in C and is licensed under the MIT License.
